# Automation Strategy (Basic)

## Objective
Automate critical flows to ensure stability and fast regression validation.

## Scope
- Login
- Registration
- Core user flows

## Tools
- Cypress / Selenium
- GitHub
- CI/CD (future integration)

## Strategy
- Prioritize stable and repetitive flows
- Focus on regression automation
- Avoid unstable UI automation
- Prefer API automation when possible

## Target
Build scalable automation framework in future iterations.
